Derniere mise a jour : Decembre 2024

En telechargeant, installant ou utilisant l'application Oniro, vous acceptez d'etre lie par ces Conditions Generales d'Utilisation. Si vous n'acceptez pas ces conditions, veuillez ne pas utiliser l'application.

2. Description du Service

Oniro est une application mobile d'interpretation de reves utilisant l'intelligence artificielle. Le service comprend :

  • L'interpretation textuelle de vos reves par IA
  • La generation d'images artistiques representant l'essence symbolique de vos reves
  • Un journal personnel pour sauvegarder vos reves
  • La saisie vocale pour dicter vos reves
  • Le partage sur les reseaux sociaux

Avertissement Important

Les interpretations fournies par Oniro sont generees par intelligence artificielle a des fins de divertissement uniquement. Elles ne constituent en aucun cas un avis medical, psychologique ou therapeutique. Si vous avez des preoccupations concernant votre sante mentale, consultez un professionnel qualifie.

3. Compte Utilisateur

Aucune creation de compte n'est requise pour utiliser Oniro. Un identifiant anonyme unique est automatiquement genere sur votre appareil pour :

  • Gerer vos credits d'images
  • Gerer votre abonnement Premium
  • Synchroniser vos achats avec les serveurs

Cet identifiant ne contient aucune information personnelle identifiable.
